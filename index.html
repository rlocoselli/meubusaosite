What makes Meu busao your right choice to move on Bus

This application is going to become even wider and more powerful. It provides users with features such as: bus routes, timetable and real-time bus location for some specific cities

Mobile platform

Our mobile applications are easy to handle and provide up-to-date data for most of cities.

Real Time Buses

For some cities as São Paulo, Brazil, we offer real time bus location as some providers are able to integrate their services along with ours by REST. You do not need to understand the technical stuff, however it is nice to know we can give you real-time bus position for some locations :)

Timetable Update

Our team updates timetable very often. All information is obtained from open source databases provided by bus agencies around the globe in format GTFS
